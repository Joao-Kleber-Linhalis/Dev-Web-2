LocaTudo Introdução

Olá! Seja bem-vindo ao sistema LocaTudo desenvolvido por mim na matéria de Desenvolvimento Web 2 no curso de Desenvolvedor Full Stack.
Este é um sistema de gerenciamento de uma Locatora desenvolvido em Angular e Spring Boot.
O principal objetivo deste projeto é abordar conceitos relacionados a Single Page Applications (SPAs), autenticação e autorização com Tokens JWT e muito mais. Você pode acessar o código-fonte do projeto clicando no link do Github no menu lateral.

Sinta-se à vontade para testar o sistema e deixar suas sugestões.
